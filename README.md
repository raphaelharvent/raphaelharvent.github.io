# Portfolio – Raphaël Harvent

Site statique hébergé sur **GitHub Pages**.

## Déploiement rapide

1. Crée un dépôt **public** nommé **RAGUN023.github.io** sur GitHub.
2. Ajoute tous les fichiers de ce dossier à la racine du dépôt.
3. Va dans **Settings → Pages** :
   - **Source** : *Deploy from a branch*
   - **Branch** : `main` (ou `master`) / `/root`
4. Attends 1 minute puis visite **https://RAGUN023.github.io**.

### Avec Git (optionnel)
```bash
git init
git add .
git commit -m "Portfolio BTS SIO SISR – initial"
git branch -M main
git remote add origin https://github.com/RAGUN023/RAGUN023.github.io.git
git push -u origin main
```

## Personnalisation
